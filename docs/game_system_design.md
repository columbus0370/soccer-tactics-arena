## 🔄 2. コアゲームループの詳細

```
┌──────────────────────────────────────────────────────────────────┐
│                    GAME LOOP (1試合 = 5分)                        │
└──────────────────────────────────────────────────────────────────┘

【フェーズ1：準備】(1分)
  ├─ マッチング / 対戦相手確定 (10-30秒)
  │   └─ PvP: ランダムマッチング or レートマッチング
  │   └─ PvE: CPU難度選択
  │
  └─ チーム編成・戦術選択 (30秒)
      ├─ フォーメーション選択（相手は見えない）
      ├─ 戦術プリセット選択
      └─ [確定]ボタン押下 → 相手の決定を待つ

【フェーズ2：試合実行】(2分)
  ├─ 初期キックオフ
  ├─ AI判定による試合シミュレーション
  │   ├─ フォーメーション相性 × 戦術相性 × ランダム要素
  │   ├─ 前半 (1分)
  │   ├─ ハーフタイム (スタッツ更新)
  │   └─ 後半 (1分)
  ├─ リアルタイム進行表示
  │   ├─ ピッチの動きアニメーション
  │   ├─ スコア更新
  │   └─ 実況テキスト表示
  └─ 最終スコア確定

【フェーズ3：リザルト】(1分)
  ├─ 最終スコア表示
  ├─ MVP発表
  ├─ 詳細統計表示
  ├─ 獲得ポイント計算
  │   ├─ 勝利: +30pts
  │   ├─ 敗北: +5pts
  │   └─ ドロー: +15pts
  ├─ バッジ・アチーブメント確認
  └─ [次へ] → ロビーに戻る

═══════════════════════════════════════════════════════════════════

【ロビー】
    ↓
【マッチング】 (10-30秒)
    ↓
【準備フェーズ】 (30秒)
  フォーメーション選択 → 戦術選択 → 確定
    ↓
【試合フェーズ】 (2分)
  前半1分 → ハーフタイム → 後半1分
    ↓
【リザルト】 (1分)
  スコア表示 → 統計 → ポイント獲得
    ↓
【ロビーへ戻る】←─────┐
  ↓                    │
【次の対戦へ】─────────┘

```

## ⚙️ 3. フォーメーション相性システム

```
┌────────────────────────────────────────────────────────────────┐
│              フォーメーション相性マトリックス                     │
└────────────────────────────────────────────────────────────────┘

                VS→   4-3-3    4-2-4    5-3-2    3-5-2
↓選ぶ
4-3-3 (バランス)  50%    45%     55%      50%
   ├─ 利点: 全体バランスが良い
   ├─ 弱点: 特に強い要素がない
   └─ 適性: 初心者向け

4-2-4 (攻撃的)    55%    50%     45%      60%
   ├─ 利点: シュート数が多い、スピード重視
   ├─ 弱点: 守備が手薄い
   └─ 適性: 攻撃好きなプレイヤー

5-3-2 (守備的)    45%    55%     50%      48%
   ├─ 利点: 失点が少ない、カウンター効果
   ├─ 弱点: 攻撃機会が限定される
   └─ 適性: 堅実なプレイヤー

3-5-2 (サイド攻撃) 50%    40%     52%      50%
   ├─ 利点: サイド経由での得点が増える
   ├─ 弱点: センターが薄い
   └─ 適性: テクニカルなプレイヤー

---

相性の例：
  "4-2-4を選んだプレイヤー"vs"4-3-3を選んだプレイヤー"
  → 4-2-4の勝率は55%
  → 4-3-3の勝率は45%
  → 差は10ポイント = 戦術による有利差

  ※ただしランダム要素（±20%）があるため、
    必ず高い方が勝つわけではない

```

  相性が良い相手: 守備型
  相性が悪い相手: カウンター型

─────────────────────────────────────

【戦術2】ロングボール型 (カウンター主体)
  攻撃特性:
    ├─ パス成功率: -10%
    ├─ シュート精度: -5%
    ├─ ボール保持率: -15%
    ├─ スピード: +15%
    └─ シュート数: +10%

  守備特性:
    ├─ 守備の安定性: -5%
    ├─ インターセプト: -8%
    └─ スライディングタックル: +12%

  相性が良い相手: パス主導型
  相性が悪い相手: ポゼッション重視

─────────────────────────────────────

【戦術3】サイド攻撃型 (クロス重視)
  攻撃特性:
    ├─ クロス精度: +20%
    ├─ ウイングのドリブル: +12%
    ├─ セットプレー得点: +8%
    └─ センターFW: 決定力依存

  守備特性:
    ├─ サイド守備の強度: +3%
    └─ センター守備の安定性: -5%

  相性が良い相手: センター弱い守備
  相性が悪い相手: センター堅い守備

## 📊 5. 試合シミュレーションロジック

```
┌──────────────────────────────────────────────────────────────┐
│           試合結果決定のAIアルゴリズム (疑似コード)              │
└──────────────────────────────────────────────────────────────┘

// ===== 初期化フェーズ =====
player1_formation = "4-3-3"      // プレイヤー1の選択
player1_tactic = "パス主導型"
player2_formation = "4-2-4"      // プレイヤー2の選択
player2_tactic = "ロングボール"

// ===== 相性スコア計算 =====
formation_advantage_p1 = 
    getFormationAdvantage(player1_formation, player2_formation)
    // 例: 50 (50%の勝率)

tactic_advantage_p1 = 
    getTacticAdvantage(player1_tactic, player2_tactic)
    // パス主導型 vs ロングボール = +5 (パス主導型有利)

// ===== ランダム要素 (±20) =====
random_factor = randomInt(-20, 20)
// 例: +12 (運が良い)

// ===== 基本勝率計算 =====
final_p1_score = formation_advantage_p1 
                + tactic_advantage_p1 
                + random_factor
                + home_advantage_bonus  // ホーム利点 (将来)
                
// final_p1_score = 50 + 5 + 12 = 67

// ===== 勝敗判定 =====
if (final_p1_score > 50) {
    RESULT = "Player1 Win"
    score = simulateMatch(player1_formation, player2_formation)
    // 例: 2-1 で Player1 勝利
} else if (final_p1_score < 50) {
    RESULT = "Player2 Win"
    score = simulateMatch(player2_formation, player1_formation)
} else {
    RESULT = "Draw"
    score = [1, 1]
}

// ===== 詳細スタッツ生成 =====
stats = {
    shots: randomInt(8, 15),
    shots_on_target: randomInt(3, 7),
    pass_accuracy: 50 + random(-15, 15),
    possession: 50 + random(-20, 20),
    tackles: randomInt(15, 25),
    intercepts: randomInt(5, 15),
    crosses: randomInt(10, 20),
    fouls: randomInt(8, 15),
}

// ===== リザルト =====
{
  result: "Player1 Win",
  final_score: "2-1",
  stats: { ... },
  mvp: "FW太郎",
  points_earned: 30  // 勝利ボーナス
}

```
