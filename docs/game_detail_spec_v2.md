## 1. 選手システム詳細

### 1.1 選手プール

- **総数：** 200名以上（MVP時点で最低200名用意）
- **管理方式：** プリセット固定リスト。全ユーザーが同じ選手を参照する
- **名前：** スキッパー名（架空の実名風。例：「田中 翔」「Carlos Mendez」）
- **育成：** なし。能力値はゲームを通じて変動しない

### 1.2 選手能力値システム

各選手は以下の**6つの基本能力値**を持つ。値は1〜100の整数。

| 能力値 | 説明 | 主に影響するポジション |
|--------|------|----------------------|
| スピード | 移動・カウンター速度 | FW・MF（サイド） |
| シューティング | 得点力・シュート精度 | FW |
| パス | パス成功率・展開力 | MF |
| ドリブル | 個人突破力 | FW・MF |
| フィジカル | 競り合い・プレス耐性 | DF・MF |
| ディフェンス | タックル・クリア成功率 | DF・GK |

**ポジション別補正（乗数）：**

| ポジション | シューティング | パス | ディフェンス | スピード |
|-----------|--------------|------|------------|---------|
| GK | ×0.2 | ×0.5 | ×1.5 | ×0.5 |
| DF | ×0.4 | ×0.8 | ×1.5 | ×0.9 |
| MF | ×0.8 | ×1.5 | ×0.9 | ×1.0 |
| FW | ×1.5 | ×0.7 | ×0.3 | ×1.3 |

**選手ランク（能力値の合計帯域）：**
- **Sランク：** 合計450以上（エース級・希少）
- **Aランク：** 合計380〜449（主力）
- **Bランク：** 合計310〜379（レギュラー）
- **Cランク：** 合計250〜309（控え）

## 2. チーム編成システム詳細

### 2.1 編成の自由度

- 11人の各スロット（GK×1、DF×可変、MF×可変、FW×可変）に対して
  プリセット選手リストから個別に選手を指定できる
- 同じ選手を複数スロットに入れることは不可
- ポジション外の選手を起用可能だが、**ポジション補正がかからないペナルティ**がある

### 2.2 フォーメーション選択画面のUI

```
   ┌────────────────────────────────┐
   │        ピッチ俯瞰図             │
   │                                │
   │           [GK ●]               │
   │                                │
   │    [DF ●]  [DF ●]  [DF ●]     │
   │                                │
   │  [MF ●]  [MF ●]  [MF ●]      │
   │                                │
   │      [FW ●]  [FW ●]           │
   │                                │
   │  ← 4-3-2 フォーメーション →   │
   └────────────────────────────────┘
   
   ● = 選手ドット（タップで選手詳細表示・交代可能）
```

### 2.3 フォーメーション種類（MVP時点）

| フォーメーション | 特徴 | 推奨戦術 |
|---------------|------|---------|
| 4-3-3 | バランス型（デフォルト） | パス主導 |
| 4-2-4 | 攻撃偏重、守備薄 | カウンター |
| 5-3-2 | 守備重視、カウンター有効 | 守備固め |
| 3-5-2 | サイド攻撃型 | サイド展開 |
| 4-4-2 | オーソドックス、初心者向け | なんでも |

### 2.4 チーム編成プリセット保存

- 最大**5つのプリセット**を事前に保存可能
- 各プリセットに名前をつけられる（例：「攻撃型A」「守備型B」）
- 試合開始前にプリセットを呼び出し、その場で微調整も可能

## 3. 戦術システム詳細

### 3.1 戦術の深さ（確定）

**中程度レベル：**
- 良い戦術を使えば勝率が上がる（約60%まで）
- ただし初心者でも運と選手力で勝てる（最低40%は保証）
- 戦術相性の知識を学ぶほど有利になるが、覚えなくてもプレイできる

### 3.2 試合前の戦術設定

**ステップ1：フォーメーション選択**（上記2.3参照）

**ステップ2：攻撃戦術の選択**

| 攻撃戦術 | 効果 | 相性の良い相手 | 弱点 |
|---------|------|--------------|------|
| パス主導型 | パス成功率+15%、ポゼッション+20% | 守備固め相手 | カウンターに弱い |
| カウンター | 速攻成功率+20%、体力消費少 | パス主導型相手 | ポゼッションで負ける |
| サイド展開 | クロス精度+20% | センター守備が堅い相手 | サイド守備が薄い相手に弱い |
| セットプレー重視 | FK・CK得点率+25% | 守備が硬い相手 | 流れからの得点が少ない |

**ステップ3：守備戦術の選択**

| 守備戦術 | 効果 | 相性の良い相手 | 弱点 |
|---------|------|--------------|------|
| ハイプレス | インターセプト+20%、体力消費大 | パス主導型相手 | スタミナが後半に響く |
| リトリート | 失点リスク-15%、カウンター起点 | 攻撃的相手 | ポゼッション率が低下 |
| マンマーク | エース封じ、1対1に強い | エース依存チーム | 組織的な攻撃に弱い |
| ゾーンディフェンス | 安定したブロック形成 | バランス型チーム | セットプレーに弱い |

### 3.3 試合中の介入システム（確定）

試合中に**最大2回**、大まかな方向性の指示を出すことができる。

**指示できる内容：**

| 指示 | 効果 | 使うべき場面 |
|-----|------|------------|
| 攻撃強化 | シュート数+30%、失点リスク+10% | 負けている時 |
| 守備固め | 失点リスク-20%、攻撃力-15% | リードしている時 |
| 右サイド攻撃 | 右サイドからのクロス+25% | 相手左サイドが弱い時 |
| 左サイド攻撃 | 左サイドからのクロス+25% | 相手右サイドが弱い時 |
| 均等プレー | 元の戦術設定に戻す | 指示をリセットしたい時 |

**UI設計：**
- 試合中画面の下部に「指示ボタン」を設置
- 残り指示回数をバッジで表示（例：「指示 2/2」→「指示 1/2」）
- 指示を出した後30秒間はクールダウン（連打防止）

## 4. 試合の演出システム詳細

### 4.1 試合中アニメーション

試合中はピッチ俯瞰図上で以下をリアルタイムに表示：

**常時表示：**
- スコア（リアルタイム更新）
- 試合時間（0〜90分をゲーム内2分で表現）
- ポゼッション率バー

**イベント発生時：**
- シュート → ゴール前にドットが集まるアニメーション
- 得点 → スコアがアニメーションで更新＋得点者名表示
- 大きなチャンス → 画面が一瞬フラッシュ
- ファール/カード → 小アイコンで表示

**スキップ機能：** 試合アニメーションはいつでも「早送り」または「スキップ」できる

### 4.2 得点シーン詳細表示

得点が入った際に以下を表示するポップアップ：

## 6. ゲームバランス設計

### 6.1 勝率への影響割合

```
試合結果 = 
    フォーメーション相性 × 25%
  + 戦術相性（攻守）    × 25%
  + チーム総合能力値    × 30%
  + 試合中指示の効果    × 10%
  + ランダム要素        × 10%
```

**想定される結果の幅：**
- 最大有利差：60対40（戦術と能力値が両方有利な場合）
- 実力差が拮抗している場合：55対45程度
- ランダム要素で逆転可能な範囲を常に確保

### 6.2 初心者向け配慮

- **おすすめ表示：** フォーメーション選択時に「初心者向け ★」マーク
- **ヒント表示：** 戦術選択時に「このフォーメーションには〇〇戦術が合います」とヒント
- **チュートリアル：** 初回プレイ時に1試合分のガイド付きプレイ

以下は今回のヒアリングで未確定だった項目です。次のステップで決定が必要です。
